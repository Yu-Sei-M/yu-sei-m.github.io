PWAとは、プログレッシブウェブアプリケーションと呼ばれるGoogleが提起した規格に沿ったウェブサイト、ウェブアプリのことです。PWAもSPAとは全く異なる理由で表示が高速で、SEOの観点でも非常に有利である（検索して上位表示されやすくなる）他、iOSのAppStoreやAndroidのPlayStoreにアプリを公開することなくウェブアプリをスマホアプリのようにユーザーに使ってもらうことができる（３０％の手数料を回避できる）などの特長もあって注目を集めています。第３回では、ハンズオンで作成したアプリをさらにカスタマイズした上でPWAを完成させることを目指します。
