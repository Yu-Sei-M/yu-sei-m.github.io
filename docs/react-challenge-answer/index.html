課題１：フッターコンポーネント（footer.js）を作成して、トップページにフッターを追加してください。クラス名はApp-footerとし、背景色は「#efefef」とします。

コンポーネントディレクトリの中にフッター用のファイルを作成しましょう。

コマンドラインでファイル作成

touch src/components/footer.js

作成できたら、VSCodeで開き、以下のように記載します。

footer.js

const Footer = () => <footer className="App-footer">©️2020 NPO法人FitnessEngineer</footer>

export default Footer;

footerタグのクラスに「App-footer」と名前をつけました。ここはCSSと対応していれば名前はなんでも大丈夫です。

App.css

.App-footer {
  background-color: #efefef;
  position: absolute;
  bottom: 0;
  width: 100%;
  padding: 16px 0px;
}

上記のようにApp.cssにフッター用のスタイリングを記述しましょう。

最後にApp.jsに読み込みの記述と呼び出しをかけば完了です。

App.js

// 三行目に追記
import Footer from './components/footer.js'

// </header>の次の行に追記
<Footer/>

上記の変更を保存してブラウザを確認すると、以下のようになっていますね。

完成後の状態

解説：HTMLで習ったタグを書いて、classNameに自由にクラス名をつけ、そのクラス名にCSSでスタイルを書く流れを覚えておきましょう。

以下のように移動してドロップ

const logo = "/images/画像のファイル名"
