このページではMacの方向けの説明をしています。

このページでやること

  • Homebrewのインストール
  • Nodeのインストール
  • Nodeのバージョン管理ツールのインストール
  • Nodeのバージョンを変更

Homebrewがない人はインストールしましょう。

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

ターミナルで次のコマンドを実行しましょう。

brew install node

完了したら次のコマンドを実行しましょう。

npm -g i n

続いて、次のコマンドを実行しバージョンを変更します。

sudo n 10.19.0

最後に正しくバージョンが変更できているかチェックしておきましょう。

sudo node -v
