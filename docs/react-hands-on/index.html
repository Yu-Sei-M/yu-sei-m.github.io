指示１：背景色をカラーコード「#fefefe」に変更し、見出し文字を「dimgrey」に変更しましょう。

一行目のバックグラウンドカラーをコメントアウトして値を「background-color: #fefefe;」を追記します。同様に、最後のカラーをコメントアウトして値を「color: dimgrey」にしましょう。

App.cssの中のApp-headerクラスのスタイルを変更

.App-header {

  /* （中略） */
  /* color: white; */

指示２：リンクの色をカラーコード「#294b8f」に変更しましょう。

}

const App = () => {
  const logo = '/images/logo.png';
  return (
    <div className="App">
      <header className="App-header">

        </p>

    </div>

}
