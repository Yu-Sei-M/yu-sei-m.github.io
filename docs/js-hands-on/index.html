指示１：デスクトップにフォルダを作成し、その中にjs.htmlを作成しましょう。

ターミナルでやる人はこちら

cd ~/Desktop

touch js.html

指示２：実行が完了したら、VSCodeでjs.htmlを開きましょう。

指示３：次のコードを記述してください。

<script>
  console.log(document)
</script>

出力結果を一緒に確認しましょう！

指示４：次のコードを記述してください。

<script>
  window.addEventListener('load', () => {
    console.log(document)
  });
</script>

このようにしてロードが完全に終了した後でなければ、要素を操作することができません。

headの中に１つのstyleタグと、bodyの中に４つのdivタグがあるので、作成しましょう。

<script>
  window.addEventListener('load', () => {
    console.log(document)
    var style = document.createElement('style');
    document.head.appendChild(style);
    var header = document.createElement('div');
    document.body.appendChild(header);
    var menu = document.createElement('div');
    document.body.appendChild(menu);
    var contents = document.createElement('div');
    document.body.appendChild(contents);
    var footer = document.createElement('div');
    document.body.appendChild(footer);
  });
</script>

課題１：先ほどdivタグのうち、menuとcontentsを残しておきました。残り時間に余裕のある方は、ぜひ自分の頭で考えて完成させてみましょう。

  • menuの中にアンカータグを４つ配置する
  • contentsの中の複雑な構造を再現する
  • その上でお題のHTML・CSSを再現する
